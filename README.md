# Impact of Crude Oil Prices on India's Fiscal and Current Account

## Project Overview

India imports more than 80% of its crude oil demand, making the economy highly sensitive to global oil price shocks.

This project investigates the econometric relationship between:

• Brent crude oil prices  
• India's Gross Fiscal Deficit (GFD)  
• India's Current Account Deficit (CAD)

The analysis uses historical data from 1971–2024.

## Methodology

The econometric analysis was conducted using STATA.

Steps followed:

1. Simple Linear Regression
2. White Test for heteroscedasticity
3. Log transformation of variables
4. Durbin Watson test for autocorrelation
5. Prais–Winsten transformation to correct AR(1)

## Data Sources

• Reserve Bank of India (RBI)  
• World Bank Commodity Database

## Key Results

Fiscal Deficit Model
Adjusted R² ≈ 0.36

Current Account Model
Adjusted R² ≈ 0.08

Results indicate that crude oil prices significantly influence India's fiscal balance and external sector.

## Tools Used

• STATA  
• Econometrics  
• Time series macroeconomic data
